🧪 Testanweisungen

Schritt 1: System öffnen

  1. Öffne die Hauptanwendung (index.html)
  2. Klicke im Seitenmenü auf "Lieferantenbewertung"
  3. Das Dashboard sollte mit 3 Beispiel-Lieferanten angezeigt werden
  4. Überprüfe die Statistiken und Kennzahlen

Schritt 2: Neuen Lieferanten hinzufügen

  1. Klicke auf "Neuer Lieferant"
  2. Navigiere durch alle 5 Tabs des Formulars
  3. Fülle alle Pflichtfelder aus
  4. Teste die Form-Validierung
  5. Speichere den Lieferanten

Schritt 3: Filterung testen

  1. Wechsle zum Tab "Lieferanten"
  2. Teste alle Filter-Optionen (Status, Typ, Suche)
  3. Überprüfe die Echtzeit-Filterung
  4. Teste die Kombinationen verschiedener Filter

Schritt 4: Quick Actions

  1. Teste "Bewertung durchführen"
  2. Teste "Zertifikate prüfen"
  3. Teste "Audit planen"
  4. Überprüfe die Konsolen-Ausgabe für Debug-Meldungen
